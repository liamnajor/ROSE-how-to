Object Editing

Adding Objects

To add an object, change the mode selector to "Object", set the ID to whatever value you want, with 2 caviets: if you want the object to not respawn, set it between 40 and 7F, and try to avoid sharing ID's with multiple items on one screen. Deleting Objects

to delete an object, locate which one you wish to delete via the "View Object Data" button, select it, via the selector right next to the "delete object" button, then press said button.
